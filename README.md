# 🚀 LabelForge  

Ένα απλό εργαλείο για **labeling κειμένων** (θετικά / αρνητικά / ουδέτερα).  

---

## ✨ Λειτουργίες
- 📂 Φόρτωση κειμένων για αξιολόγηση  
- 🏷️ Επιλογή ετικέτας (Positive, Neutral, Negative)  
- 📤 Εξαγωγή δεδομένων σε **CSV** και **JSONL**
